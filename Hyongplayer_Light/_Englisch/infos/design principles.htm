Important issues to be balanced:

  • portability (runnable on each operating system with Java: Windows, MacOS, Linux, ...)
    Consequence: no special characters and vowel mutation
  • easy to use and appropiate user interface (normal and expert mode)
  • balanced data interface (adequate interface for individual steps sequences)
  • clearly arranged implementation
  • performance (smooth picture flow)
  • stability and rubustness (faulty data are ignored)
